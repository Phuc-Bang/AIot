Project này làm gì?

Project mô phỏng một pipeline AIoT hoàn chỉnh: tải dữ liệu nhiệt độ, gắn label anomaly, tạo feature theo thời gian, train model, sinh event log và deploy API nhận telemetry mới.

Vì sao cần anomaly detection?

Trong AIoT, một giá trị đơn lẻ hiếm khi đủ để kết luận. Cùng là 28 độ C, nó có thể bình thường vào buổi trưa nhưng bất thường nếu vừa tăng đột ngột từ mức nền 20 độ C.

Tư duy chính Model phát hiện tín hiệu bất thường. Backend phải biến tín hiệu đó thành event, severity và decision an toàn.

Key Artifacts

models/isolation_forest_iforest_v1.joblib Model Isolation Forest chính được FastAPI load để score telemetry mới.
outputs/iforest_metrics.json Metric offline: Precision, Recall, F1, Confusion Matrix và threshold demo.
outputs/anomaly_event_log.csv Event log có event_type, severity, decision và explanation.
figures/anomaly_detection_result.png Biểu đồ giúp demo điểm anomaly trên chuỗi nhiệt độ.
API /detect-anomaly Endpoint nhận history telemetry và trả JSON gồm model output, event và latency.
