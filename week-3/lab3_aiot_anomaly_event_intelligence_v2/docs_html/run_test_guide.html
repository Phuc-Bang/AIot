Lab 3 - Anomaly Detection & Event Intelligence for AIoT

Chạy project từ đầu đến API.

Thứ tự chạy đúng là data -> train -> plot -> API -> test. API load model lúc start, nên train xong cần restart API nếu trước đó API đã chạy.

Quick start

cd E:\AIoT\Day-3\lab3_aiot_anomaly_event_intelligence_v2
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python src/download_data.py
python src/train_anomaly.py
python src/plot_results.py

Các bước kiểm tra

Tạo môi trường
python -m venv .venv và activate bằng PowerShell.
Cài thư viện
pip install -r requirements.txt.
Tải dữ liệu
python src/download_data.py.
Train model
python src/train_anomaly.py.
Vẽ biểu đồ
python src/plot_results.py.
Chạy API
uvicorn src.app:app --reload.
Test API
python src/test_api.py hoặc python src/test_api_local.py.
