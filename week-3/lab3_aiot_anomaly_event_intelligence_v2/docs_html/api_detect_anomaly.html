API biến model thành service.

src/app.py load model Isolation Forest và expose ba endpoint: /health, /model-info, /detect-anomaly.

FastAPI Anomaly Score Event Type Decision

GET /health

Kiểm tra API chạy và model đã load chưa.

GET /model-info

Trả model version, input/output contract và metric offline.

POST /detect-anomaly

Nhận history telemetry, score điểm mới nhất và trả event JSON.

Request JSON

{
  "history": [
    {
      "timestamp": "2013-07-05 09:00:00",
      "value": 27.5,
      "device_id": "room_temp_01"
    },
    {
      "timestamp": "2013-07-05 09:05:00",
      "value": 29.1,
      "device_id": "room_temp_01"
    }
  ]
}

Response JSON

{
  "model_output": {
    "anomaly_score": 0.6123,
    "is_anomaly": true,
    "model_version": "iforest_v1"
  },
  "event": {
    "event_type": "TEMPERATURE_ANOMALY",
    "device_id": "room_temp_01",
    "timestamp": "2013-07-05 09:05:00",
    "value": 29.1,
    "severity": "MEDIUM",
    "decision": "CREATE_WARNING_EVENT",
    "explanation": "model score indicates unusual pattern",
    "safety_note": "Không tự động điều khiển thiết bị khi anomaly cao; cần xác nhận hoặc rule an toàn."
  },
  "api_check": {
    "latency_ms": 12.34,
    "input_points": 40
  }
}

Flow xử lý trong API

Nhận history từ request và chuyển thành DataFrame.
Parse timestamp, sort theo thời gian và tạo feature bằng add_time_features().
Lấy dòng mới nhất để score bằng pipeline scaler + Isolation Forest.
Chuyển raw score thành score demo, so sánh threshold 0.55.
Gọi decision_from_score() để tạo severity, decision và explanation.
Lỗi thường gặp model_loaded: false nghĩa là chưa chạy python src/train_anomaly.py hoặc cần restart API sau khi train.
Test nhanh Dùng Swagger UI tại http://127.0.0.1:8000/docs hoặc chạy python src/test_api.py.
