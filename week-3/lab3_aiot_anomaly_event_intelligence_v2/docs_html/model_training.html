Model output

Scikit-learn trả score càng thấp càng bất thường, nên code đảo dấu để dễ đọc.

raw_scores = -detector.score_samples(...)
anomaly_score = min_max_normalize(raw_scores)
is_anomaly = anomaly_score >= threshold

Threshold

Lab chọn threshold bằng quantile 0.92 trên anomaly score của tập test, nghĩa là nhóm score cao nhất sẽ bị đánh dấu anomaly.

Production note Không nên dùng test set để chọn threshold. Nên tách validation set hoặc calibration set theo thời gian.
