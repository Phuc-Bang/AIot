Review file chính

File / thư mục Vai trò Input Output Phụ thuộc
data/ Lưu telemetry và label. NAB hoặc sample fallback. CSV có timestamp, value, label. Training, notebook, API tests.
notebooks/ Hướng dẫn học từng bước. Data và utility. Model, metrics, chart, event log. src/utils.py.
src/download_data.py Tải data NAB và gắn label window. GitHub raw data, combined_windows. ambient_temperature_system_failure_labeled.csv. Training pipeline.
src/utils.py Core utility: load, feature, split, metrics, events. CSV hoặc DataFrame telemetry. Feature DataFrame, metrics dict, event DataFrame. Train script, API, notebook.
src/train_anomaly.py Train Isolation Forest và autoencoder demo. Feature DataFrame. Model joblib, metrics, predictions, event log. utils.py, data.
src/app.py FastAPI deploy model. Model joblib, request JSON. Response JSON có model output và event. Model đã train, utils.py.
src/test_api.py Smoke test API qua HTTP. API đang chạy, 40 dòng cuối dataset. In response của health, model-info, detect. Uvicorn.
src/test_api_local.py Test API bằng FastAPI TestClient. App Python trực tiếp. Assert schema response. src/app.py.
src/plot_results.py Vẽ kết quả anomaly. iforest_test_predictions.csv. 2 hình PNG trong figures/. Train script đã chạy.
requirements.txt Khai báo dependency. Không có. Môi trường Python chạy được lab. Tất cả script.
Nhận xét kỹ thuật src/utils.py là contract giữa training và API. Nếu sửa feature ở đây, model cũ không còn tương thích.
Đề xuất cải tiến Tách scoring và event rule thành module riêng để training, API và test dùng chung một logic.
