🔋 HỆ THỐNG TÀI LIỆU KỸ THUẬT: LAB 4 FORECASTING

Chào mừng các bạn đến với cổng thông tin tài liệu kỹ thuật toàn diện của dự án Lab 4: Dự báo và Phân tích Dự đoán Điện năng tiêu thụ (UCI Appliances Dataset). Hệ thống tài liệu này được chuyển đổi tự động từ Markdown sang HTML di động, giúp tối ưu hóa việc nghiên cứu và học tập trực quan.

🗺️ Sơ đồ Lộ trình Học tập (14 Học phần)

00. Tổng quan dự án

Giới thiệu bài toán dự báo, bối cảnh smart home, và đối sánh bản chất với Lab 3 Anomaly Detection.

01. Kiến trúc hệ thống AIoT

Kiến trúc 8 lớp của hệ thống AIoT, nơi AI sinh sống, ánh xạ BEMS tòa nhà và tích hợp ESP32/MQTT.

02. Luồng dữ liệu tuần tự

Chi tiết luồng đi của dữ liệu thô qua 15 bước biến đổi thành kết quả API dự đoán Wh.

03. Luồng thực thi & Phụ thuộc

Đồ thị dependency của hệ thống, luồng thực thi trong RAM và cẩm nang debug lỗi KeyError/NaN.

04. Tập dữ liệu & Điền khuyết

Giải nghĩa 29 cảm biến trong dataset UCI và cơ chế phòng vệ điền khuyết thiếu 2 lớp an toàn.

05. Kỹ nghệ đặc trưng thời gian

Cơ chế toán học đằng sau đặc trưng chuỗi thời gian: Lag, Rolling, Delta, Cyclic Sin/Cos và chống rò rỉ.

06. Mô hình hóa & Đo lường sai số

Tầm quan trọng của Baselines và ý nghĩa lưới điện vật lý của các sai số hồi quy MAE, RMSE, MAPE, Bias.

07. Tầng ra quyết định & Cảnh báo

Tầng ra quyết định ánh xạ rủi ro Wh sang phân vị xác suất thống kê (70%/90%/97%) và log forecast.

08. Triển khai API FastAPI

Triển khai FastAPI online serving, Sequence Diagram, JSON payloads mẫu và tích hợp Dashboard.

09. Đánh giá chất lượng mã nguồn

Đánh giá chất lượng code theo chuẩn Senior (modularity, SoC) và sơ đồ tái cấu trúc mô-đun hóa.

10. An toàn & Cơ chế Fail-safe

8 rủi ro hệ thống vật lý và giải pháp Edge Safety Gateway nhúng ESP32 bằng C++ Safe Interlock.

11. Lộ trình phát triển & Học tập

Đường dẫn học tập phễu 4 giai đoạn cho học viên và lộ trình 3 cấp độ MLOps cho doanh nghiệp.

📖 Sách hướng dẫn kỹ thuật (Handbook)

Cuốn sách hướng dẫn kỹ thuật (Handbook) tổng hợp đầy đủ từ thuật ngữ chuyên ngành đến FAQ & troubleshooting.

🧪 Hướng dẫn chạy thử nghiệm nhanh

Để khởi chạy dự án và kiểm thử dịch vụ API dự báo phụ tải Wh của ngôi nhà:

# 1. Tải và cài đặt các thư viện phụ thuộc
pip install -r requirements.txt

# 2. Tải tập dữ liệu thô UCI Appliances về máy
python src/download_data.py

# 3. Huấn luyện 5 mô hình và đóng gói model bundle tốt nhất
python src/train_forecast.py

# 4. Khởi chạy FastAPI Server serving cục bộ
uvicorn src.app:app --reload

# 5. Kiểm thử gọi API thời gian thực
python src/test_api_local.py
